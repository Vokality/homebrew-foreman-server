# Vokality Homebrew Tap

Homebrew tap for `foreman-server`.

## Install

```bash
brew tap vokality/foreman-server
brew install foreman-server
```

Direct install without a separate `brew tap` step also works:

```bash
brew install vokality/foreman-server/foreman-server
```

## Run

```bash
brew services start foreman-server
```

## Update

```bash
brew update
brew upgrade foreman-server
```

## Source

The main project lives in [Vokality/foreman](https://github.com/Vokality/foreman).
Release archives for this tap are sourced from the server releases in that repository.
